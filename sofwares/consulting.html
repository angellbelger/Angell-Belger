Consulting Criminal

O Software Consulting Criminal é um software que tem como objetivo o armazenamento da Notitia Criminis (Notícia de Crime), foi desenvolvido pela Intelligence Corporation, tendo como CEO o Sr. Angell Belger, responsável e idealizador do projeto.

O software possui uma interface intuitiva, tendo opções como: ver todos os registros; registrar ocorrência; fazer uma busca inteligente; e ver metadados dos registros.

Made in Osternack by Angell Belger
